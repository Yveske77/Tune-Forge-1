# Suno Modular Prompt Generator (React + Vite)

Includes:
- Rule-driven Suno compiler: `[ ]` hard params + sections, `( )` nuance, `{ }` reserved vars, emphasis tags
- Modular UI blocks: Meta / Architecture / Nuance / Arrangement / Vars / Lyrics
- Drag & drop arrangement builder (dnd-kit)
- FX dictionary (toggle chips) + tag pickers w/ quick suggestions
- Validation guardrails + length counters (style 4500, lyrics 5000)
- Profile defaults + history + import/export + asset download (local-only)

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
